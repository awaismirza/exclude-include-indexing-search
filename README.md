# Folder Include/Exclude Manager

Folder Include/Exclude Manager is a lightweight Visual Studio Code extension that gives you quick, intuitive control over which folders are included or excluded from workspace search and indexing. Instead of manually editing settings.json, you can now manage search visibility directly from the Explorer with a simple rightâ€‘click.

## Features
- Right-click any folder in the Explorer to include or exclude it from workspace search.
- No need to manually edit settings.json.
- Fast toggling for search visibility and indexing.

## Usage
1. Open the Explorer view in VS Code.
2. Right-click a folder.
3. Choose "Exclude Folder from Search" or "Include Folder in Search" from the context menu.

## Requirements
No additional requirements. Works in any workspace.

## Extension Settings
This extension does not add custom settings. It updates the built-in `search.exclude` workspace setting.

## Known Issues
- Only works for folders inside the workspace.
- Does not affect global settings.
